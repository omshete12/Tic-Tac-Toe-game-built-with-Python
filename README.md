# Tic Tac Toe Game (Python)

A command-line Tic Tac Toe game built using Python.  
This project demonstrates core programming logic and game flow handling.

## Features
- Two-player game
- Turn-based gameplay
- Win and draw detection
- Replay option

## Technologies Used
- Python 3
- Functions & conditionals
- Loops and user input handling

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the game:

```bash
python tic_tac_toe.py
